
CONFIG OPTIONS
	coreData
	    Should generate core data models (Default: false)

	classPrefix
	    prefix for generated classes (convention: Abbreviation of pod name e.g. `HN` for `HackerNews`).` (Default: SWG)

	podName
	    cocoapods package name (convention: CameCase). (Default: SwaggerClient)

	podVersion
	    cocoapods package version. (Default: 1.0.0)

	authorName
	    Name to use in the podspec file. (Default: Swagger)

	authorEmail
	    Email to use in the podspec file. (Default: apiteam@swagger.io)

	gitRepoURL
	    URL for the git repo where this podspec should point to. (Default: https://github.com/swagger-api/swagger-codegen)

	hideGenerationTimestamp
	    hides the timestamp when files were generated (Default: true)
